1. Overview

The cli unit is a small argv parser built on top of os.args. Users declare a spec (flags, options, positional arguments, each with help text), hand it to cli.parse with the argv list, and get back a parsed object keyed by argument name. Errors come back as clierror variants -- missing required options, unknown flags, malformed values, and so on.

Accessed via qualified names only -- cli.spec, cli.parse ...; no unqualified re-export.
